1. Iterator删除当前元素
```java
Iterator it = list.iterator();
int index = 0;
while (it.hasNext())
{
    Object obj = it.next();
    if (needDelete(obj))  //needDelete返回boolean，决定是否要删除
    {
        //todo delete
        it.remove();
    }
    index ++;
}
```
应该执行的是it.remove()方法
